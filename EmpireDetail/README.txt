# React Pack: Detail Empire FMG

Este paquete contiene `src/` y `public/` listos para pegar en tu proyecto Vite (React).
Pasos rápidos:

1) Copia la carpeta `public/assets` dentro de tu proyecto (mantén la misma ruta).
2) Copia la carpeta `src` y el archivo `index.html` del pack dentro de la raíz del proyecto Vite (reemplaza si te pregunta).
3) Ejecuta:
   ```bash
   npm install
   npm run dev
   ```
4) Si usas rutas distintas, ajusta los paths de imágenes a empezar con `/assets/...`.

> Tip: Todo el CSS de Bootstrap + estilos del tema están en `src/styles.css`.
